# AI 100 Midterm Project
Handwritten Digit Classification using CNN

Author: Keron Sidhu

## Project Overview
This project implements a Convolutional Neural Network (CNN) to classify handwritten digits using the MNIST dataset.

## Dataset
MNIST handwritten digit dataset (0–9)

## Model
Convolutional Neural Network using TensorFlow/Keras.
